## Jak používat tyto materiály

### Na workshopu

Lektor prochází sekce postupně. Kouči obcházejí účastníky a pomáhají s problémy. Každá sekce obsahuje příkazy k vyzkoušení a cvičení.

### Samostudium

Procházej sekce od začátku. Každá navazuje na předchozí – přeskakovat se nedoporučuje. Příkazy si aktivně zkoušej, nečti jen pasivně.

## Předpoklady

- Počítač (Windows, macOS nebo Linux)
- Nainstalovaný Git – pokud ne, viz [sekce Úvod](01-uvod/README.md)
- Základní orientace v příkazové řádce (jak otevřít terminál, jak se pohybovat ve složkách)
