# 5. GitHub

Dosud jsme pracovali čistě lokálně. Teď se podíváme na GitHub – nástroj, který umožňuje sdílet kód s ostatními a spolupracovat na projektech.

---

## Git vs. GitHub

Toto je jeden z nejčastějších zmatků pro začátečníky:

| | Git | GitHub |
|---|---|---|
| **Co je to?** | Verzovací systém (software) | Webová platforma pro hostování repozitářů |
| **Kde běží?** | Na tvém počítači | Na internetu (servery GitHubu) |
| **Kdo ho vytvořil?** | Linus Torvalds, 2005 | GitHub Inc., 2008 (dnes Microsoft) |
| **Je zdarma?** | Ano, open source | Zdarma pro veřejné repozitáře |

**Git** je nástroj – funguje i bez GitHubu. **GitHub** je jen jedno z míst, kde můžeš svůj Git repozitář hostovat.

### Alternativy ke GitHubu

Existují i jiné platformy, které fungují stejně:

- **GitLab** – [gitlab.com](https://gitlab.com) – populární alternativa, open source verze lze hostovat sám
- **Bitbucket** – [bitbucket.org](https://bitbucket.org) – od Atlassianu, integrace s Jirou
- **Codeberg** – [codeberg.org](https://codeberg.org) – nezisková alternativa

---

## Registrace na GitHubu

1. Jdi na [github.com](https://github.com)
2. Klikni na **Sign up**
3. Vyplň uživatelské jméno, email a heslo
4. Dokonči registraci

### Uživatelský profil

Po přihlášení uvidíš svůj profil na adrese `github.com/tvoje-uzivatelske-jmeno`.

Na profilu jsou vidět:
- **Repositories** – seznam tvých repozitářů
- **Contributions** – graf aktivity (commitů za poslední rok)
- **Pinned repositories** – repozitáře, které chceš zvýraznit

---

## Open source

**Open source** znamená, že zdrojový kód projektu je veřejně dostupný – každý si ho může prohlédnout, použít, upravit a šířit dál (podle podmínek licence).

### Proč přispívat do open source?

- Učíš se od ostatních – vidíš, jak píší kód zkušenější vývojáři
- Získáváš zpětnou vazbu na svůj kód
- Pomáháš komunitě a projektům, které sám/a používáš
- Buduje se ti portfolio

### Příklady open source projektů na GitHubu

- [Python](https://github.com/python/cpython) – samotný interpret Pythonu
- [VS Code](https://github.com/microsoft/vscode) – editor od Microsoftu
- [Django](https://github.com/django/django) – webový framework pro Python
- [React](https://github.com/facebook/react) – JavaScriptový framework od Facebooku

Všechny tyto projekty mají na GitHubu svůj kód a kdokoli může nahlásit chybu, navrhnout změnu nebo přispět kódem.

---

## Budování portfolia

GitHub je v IT světě jako životopis pro programátory. Recruiteři a firmy se dívají na GitHub profily kandidátů.

**Doporučení:**

- Dávej svůj kód na GitHub, i když není dokonalý
- Pište popisné commit zprávy
- Přidávej `README.md` soubory s popisem projektů
- Přispívej do open source projektů

Pravidelná aktivita (zelené čtverce v grafu příspěvků) ukazuje, že programuješ i mimo práci a školu.

---

## Shrnutí

- Git ≠ GitHub – Git je nástroj, GitHub je platforma
- GitHub umožňuje hostovat repozitáře a spolupracovat s ostatními
- Existují alternativy: GitLab, Bitbucket, Codeberg
- Open source projekty jsou veřejně dostupné a každý může přispět
- Aktivní GitHub profil je cennou součástí portfolia
